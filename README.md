# 오늘의 한숨

길게 누르는 만큼 한숨의 길이를 기록하고, 오늘의 누적 한숨 시간에 따라 작은 처방을 제안하는 정적 웹앱입니다.

## 핵심 기능

- 길게 누르는 한숨 버튼
- 누르는 동안 보이는 기체 그래픽
- 오늘 한숨 횟수, 총 배출 시간, 평균 한숨 기록
- 실시간 한숨 현황처럼 보이는 라이브 피드
- 말 없이 공감하는 무언 채팅
- 누적 상태에 따른 오늘의 처방
- 결과 카드 PNG 저장 및 공유

## 실행 방법

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다.

```text
index.html
styles.css
app.js
```

## 배포 방법

정적 파일만 사용하는 앱이라 아래 서비스에 바로 올릴 수 있습니다.

- Netlify Drop: 폴더를 드래그앤드롭
- Vercel: GitHub 저장소 연결 후 배포
- GitHub Pages: 저장소 Pages 설정 활성화

배포할 때는 `index.html`, `styles.css`, `app.js`가 같은 폴더에 있어야 합니다.

## 현재 구현 메모

현재 실시간 현황은 실제 서버 데이터가 아니라 브라우저에서 생성되는 데모 데이터입니다. 실제 여러 사용자의 한숨을 모으려면 Supabase, Firebase, Cloudflare Workers 같은 작은 백엔드를 붙이면 됩니다.
